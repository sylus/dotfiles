# github.com/sylus/dotfiles

My personal dotfiles, managed with [`chezmoi`](https://github.com/twpayne/chezmoi).

## Installation

### Ubuntu / WSL

```sh
curl -sfL https://git.io/chezmoi | sh
chezmoi init --apply --verbose sylus
```

## Install location

```sh
~/.local/share/chezmoi
```
